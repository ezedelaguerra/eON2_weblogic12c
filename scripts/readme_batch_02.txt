
How to Execute Migration Scripts for New EON - BATCH 02
--------------------------------------------

1.)     Extract the rar file

2.)     Go to scripts\migration directory (cd ...\scripts\migration)

3.)     Type sqlplus.exe /nolog @02_run_Batch_02.sql


(Development sample parameters)
Enter value for var_sys_pwd: system
Enter value for var_dbname: eondev2
Enter value for var_user_pwd: eon01
Enter value for var_compile_schema: EON01
