How to Create Database for New EON
----------------------------------

1.)     Extract the rar file

2.)     Go to scripts directory (cd ...\scripts)

3.)     Type sqlplus.exe /nolog @00_run.sql




How to Execute Migration Scripts for New EON
--------------------------------------------

1.)     Extract the rar file

2.)     Go to scripts\migration directory (cd ...\scripts\migration)

3.)     Type sqlplus.exe /nolog @01_run_Batch_01.sql




How to Delete Dealing Pattern for Old EON
-----------------------------------------

1.)     Extract the rar file

2.)     Go to scripts\delete_dp directory (cd ...\scripts\delete_dp)

3.)     Type sqlplus.exe /nolog @01_delete_Batch_01.sql
